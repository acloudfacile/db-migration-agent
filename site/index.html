Quick Start

Up and Running in Minutes

Four steps from install to a fully validated migration.

Install

Clone the repo and install with pip in development mode.

pip install -e ".[dev]"

Configure

Generate a starter config and fill in your database credentials.

dbmigrate config generate --output my_migration.yaml

Assess

Run the AI assessment agent to evaluate risk and get recommendations.

dbmigrate migrate --config my_migration.yaml --assess-only

Migrate

Execute the full migration with real-time progress and validation.

dbmigrate migrate --config my_migration.yaml

Prefer a Visual Interface?

Launch the web dashboard for a full migration wizard, real-time monitoring, AI chat assistant, and interactive assessment reports.

./web/run.sh
